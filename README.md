# Crop Yield Dataset 
This repository contains crop yield data collected from smallholder farms in Kenya during the 2025 growing season. It includes information on maize, beans, and sorghum yields, along with soil quality and rainfall records.

## Dataset Features
- `Farmer_ID`: Unique identifier for each farmer
- `Crop_Type`: Maize, Beans, Sorghum
- `Yield_kg`: Harvested crop yield in kilograms
- `Soil_pH`: Soil acidity level
- `Rainfall_mm`: Rainfall recorded during the season
- `Region`: County where data was collected
  
Data was collected through field surveys and local agricultural extension reports between March–October 2025.

This dataset can be used for:
- Yield prediction modeling
- Soil and rainfall correlation analysis
- Agricultural planning and resource allocation
